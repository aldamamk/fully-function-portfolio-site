Two Sisters Bakery

Run on Wordpress, Two Sisters Bakery is a local Bakery shop with 3 branches located in the lower mainland. The store offer freshly baked good products that is sold online and in store.

Goal: The main goal of the website is to increase foot traffic andto increase sales.

Role: Involved in every part of the project, I did a portion of every development step and we set deadlines to accomplish specific tasks.

Members: Mikaela, Jenny, Gaia

Duration: ~5 weeks

Live demo Github

Content Planning

Information Architecture

The team went through the list of requirements with what the client needs for their website and planned accordingly.

Site Map

Using the list of requirements, we were able to build a site map of what pages would be needed and how everything will be placed in Wordpress.

Wireframes

We decided to do a desktop version of the wirefram since it is an ecommerce site, a lot of content would have to be put in the site and so we thought it made more sense to do the desktop version so we'll be able to envision it more easily when we are doing the mockup.

Design

Two Sisters Bakery was designed to have a homey vibe, colors are kept cool and neutral.

Mockup

Code

Before coding, the team set the globals together so everyone is on the same page and each member is given specific pages to code so as to not have issues when pulling the files into Github.

Live Site

The live site was tested in different browser and screen width devices, problem were solved before finalizing it.

Reflections:

Working in a group can help speed up the development process, we were able to brainstorm solutions when problem arises and held each other accountable for the quality of our work.
